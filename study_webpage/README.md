# HTML/CSS/JS 빠른 학습 페이지

간단한 HTML5/CSS3/JavaScript를 연습할 수 있는 로컬 웹 페이지입니다.

구성:
- `index.html` — 메인 페이지
- `css/styles.css` — 스타일 시트
- `js/script.js` — 간단한 DOM 예제, 퀴즈, 라이브 미리보기

사용 방법:
1. `c:/work/study_webpage/index.html` 파일을 브라우저로 열거나,
2. 로컬 서버를 사용하려면 Python 3가 설치된 경우 아래 명령을 실행하세요 (PowerShell에서):

```powershell
# 현재 디렉터리가 study_webpage일 때
python -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```

주의: 라이브 편집기는 학습용으로 단순 구현되어 있으며, XSS 방어는 완전하지 않습니다.
