HTML5 기초

HTML은 웹 페이지의 구조를 정의합니다. 아래는 기본 문서 구조 예시입니다.

<!doctype html>
<html lang="ko">
  <head>
    <meta charset="utf-8">
    <title>문서 제목</title>
  </head>
  <body>
    <h1>안녕하세요</h1>
  </body>
</html>

CSS3 기초

CSS는 스타일을 적용합니다. 선택자 예시:

h1 { color: #2a9d8f; }
.container { max-width: 900px; margin: 0 auto; }

JavaScript 기초

JavaScript로 DOM을 조작하고 이벤트를 처리할 수 있습니다.

document.getElementById('btn').addEventListener('click', function(){
  alert('버튼 클릭!');
});
