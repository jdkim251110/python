# Tetris (pygame)

간단한 파이썬 테트리스 게임입니다. `pygame`을 사용합니다.

요구사항

- Python 3.7+
- `pygame` (requirements.txt 포함)

설치 (Windows PowerShell 예)

```powershell
python -m pip install -r requirements.txt
```

실행

```powershell
python c:\work\test.py
```

조작

- ← / → : 이동
- ↑ : 회전
- ↓ : 소프트 드롭
- Space : 하드 드롭
- P : 일시정지
- Esc / Q : 종료

간단한 구현이므로 추가 기능(음악, 저장 점수, 더 정교한 벽킥 등)은 자유롭게 확장하세요.
